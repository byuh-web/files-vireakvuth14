  1. List out the different ways to select an item.

    2. => Elements can be selected using element selectors, class selectors, ID selectors, descendant selectors, and pseudo-selectors.

  2. What is the difference between an ID and a Class?

    3. => An ID is unique and used for one element only, while a class can be used on multiple elements.

  3. What is the difference between a SPAN tag and a DIV tag?

    4. => A SPAN is an inline element used inside text, while a DIV is a block-level element used to group larger sections.

  4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

    5. => A DIV is a generic container, while header, main, and footer are semantic tags that describe the purpose of the content.

  5. What are the different ways to select a color?

    6. => Colors can be selected using color names, hexadecimal values, RGB, or RGBA values.
